AIoT + Business

Our Goal

OnDeviceAI aims to discover best commercial uses for Smart IoT. OnDeviceAI focused to explore state of art model of vision, speech and natural language understanding and train local datasets with deep learning models then deploy pre-trained models to run efficiently on any devices. We innovate Smart IoT business models using latest deep learning technologies.

Vision

OnDeviceAI vision is to run any AI models on any devices efficiently. We explore state of the arts deep neural nets, generative and reinforcement learnings to embed machine intelligence on Smartphones, IoTs, and Robots. We focus on cutting edge use cases to improve human productivities in any industries.

Mission

OnDeviceAI mission is to reach human level intelligence on "basic recognition" for images, texts and voices. We expand those basic capabilities to generate synthetic data samples from estimated real data distributions. Later, we enable natural learning from data with reinforcement paradigm. All capabilities can run efficiently on devices.
